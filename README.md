
# 👋 (http://tae-uk.com)
새로 배운 기술들을 적용할 수 있는 기술 블로그입니다.  
현재는 개발 하면서 겪었던 내용들을 기록하고 있습니다.  
추후에는 만들고 싶은 서비스들을 개발하여 블로그에 적용할 예정입니다.

<br>

## 🛠 사용한 기술
Back
- Java
- Spring Boot
- JPA
- Query-DSL
- Redis
- PostgreSQL

Front
- React
- React-Query
- Recoil

ETC
- NCP
- Nginx
